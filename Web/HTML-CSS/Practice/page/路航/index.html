公司简介

远峰科技简介

广东远峰电子科技股份有限公司，从事于专业的嵌入式产品设计，涵盖汽车电子、可穿戴类消费电子产品，是一家涉足研发、生产、销售、物流等领域的技术导向型高科技企业。

目前公司在国内同行业中便携式导航设备市场占有率保持长期领先，手持导航开发方案在国内市场处于行业标杆地位。我们基于客户需求持续创新，目前，我们的产品和解决方案已经应用于全球5大区域或国家，即东欧、西欧、北美、南美、俄罗斯。

公司坐落于有“中国第一湿地”、“未来中国硅谷”之称的广东东莞松山湖高新技术产业开发区，地处东莞几何中心，南临香港、深圳，北靠广州，地理位置十分优越。松山湖远峰科技园是一座占地面积超过8万平方米的含写字楼、厂区、科研基地、物流园、公寓、休闲运动场馆为一体的大型综合产业基地与员工生活区，与华为科技园毗邻。
